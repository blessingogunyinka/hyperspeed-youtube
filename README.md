# Hyperspeed Youtube
Chrome extension that adds playback speeds faster than 2x (up to 3.5x) directly in the Youtube video player.

&nbsp;
&nbsp;

Currently working as of February 22nd, 2024 on Brave Browser Version 1.56.14
